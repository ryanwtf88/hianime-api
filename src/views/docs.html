Hianime Video API

A simple and powerful API to embed anime videos directly from Hianime.

API Documentation

Use our player to embed videos easily just in one step. Perfect for anime streaming websites.

GET Embed Endpoint

https://api-animo.vercel.app/api/v1/embed/{episodeId}?type={sub|dub}

Parameters

Parameter Type Required Description
episodeId String Yes The episode ID from hianime.to (e.g., 136197)
type String No Audio type: sub or dub (default: sub)

How to use

1. Go to hianime.to and open an episode.

2. Look at the URL: https://hianime.to/watch/anime-name?ep=136197

3. Copy the number after ?ep= (e.g., 136197).

4. Use it in the embed URL.

Example Code

<iframe 
  src="https://api-animo.vercel.app/api/v1/embed/136197?type=sub" 
  width="100%" 
  height="500px" 
  frameborder="0" 
  allowfullscreen>
</iframe>
