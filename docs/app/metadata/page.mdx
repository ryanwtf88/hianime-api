# Metadata

## Genres
Get a list of all anime genres.

**Endpoint**
\`GET /genres\`

**Example**
\`\`\`bash
curl http://localhost:3030/api/v1/genres
\`\`\`

**Response Schema**
\`\`\`json
{
  "success": true,
  "data": [
    { "name": "Action", "slug": "action" },
    { "name": "Adventure", "slug": "adventure" }
  ]
}
\`\`\`

---

## Characters
Get characters for a specific anime.

**Endpoint**
\`GET /characters/:id\`

**Example**
\`\`\`bash
curl http://localhost:3030/api/v1/characters/one-piece-100
\`\`\`

**Response Schema**
\`\`\`json
{
  "success": true,
  "data": [
    {
      "id": "eren-yeager",
      "name": "Eren Yeager",
      "role": "Main",
      "image": "https://cdn.noitatnemucod.net/thumbnail/100x100/100/..."
    }
  ]
}
\`\`\`

## Character Details
Get detailed information about a character.

**Endpoint**
\`GET /character/:id\`

**Example**
\`\`\`bash
curl http://localhost:3030/api/v1/character/luffy
\`\`\`

**Response Schema**
\`\`\`json
{
  "success": true,
  "data": {
    "id": "eren-yeager",
    "name": "Eren Yeager",
    "image": "...",
    "about": "...",
    "voiceActors": [...]
  }
}
\`\`\`

## News
Get latest anime news.

**Endpoint**
\`GET /news\`

**Example**
\`\`\`bash
curl http://localhost:3030/api/v1/news
\`\`\`

**Response Schema**
\`\`\`json
{
  "success": true,
  "data": [
    {
      "id": "news-id",
      "title": "News Title",
      "image": "...",
      "time": "..."
    }
  ]
}
\`\`\`
