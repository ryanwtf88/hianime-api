# Metadata

## Genres
Get a list of all anime genres.

**Endpoint**

\`GET /genres\`

**Example**

\`\`\`bash
curl http://localhost:3030/api/v1/genres
\`\`\`

**Response Schema**

\`\`\`yaml
success: true
data:
  - name: Action
    slug: action
  - name: Adventure
    slug: adventure
  - name: Comedy
    slug: comedy
  - name: Drama
    slug: drama
  - name: Fantasy
    slug: fantasy
  # ... more genres
\`\`\`

---

## Characters
Get characters for a specific anime.

**Endpoint**

\`GET /characters/:id\`

**Parameters**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| id | string | Yes | The anime ID (e.g., 'one-piece-100') |

**Example**

\`\`\`bash
curl http://localhost:3030/api/v1/characters/one-piece-100
\`\`\`

**Response Schema**

\`\`\`yaml
success: true
data:
  - id: luffy
    name: Monkey D. Luffy
    role: Main
    image: https://cdn.noitatnemucod.net/thumbnail/100x100/100/...
    cast: Mayumi Tanaka
  - id: zoro
    name: Roronoa Zoro
    role: Main
    image: https://cdn.noitatnemucod.net/thumbnail/100x100/100/...
    cast: Kazuya Nakai
\`\`\`

## Character Details
Get detailed information about a character.

**Endpoint**

\`GET /character/:id\`

**Parameters**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| id | string | Yes | The character ID |

**Example**

\`\`\`bash
curl http://localhost:3030/api/v1/character/luffy
\`\`\`

**Response Schema**

\`\`\`yaml
success: true
data:
  id: luffy
  name: Monkey D. Luffy
  image: https://cdn.noitatnemucod.net/thumbnail/300x400/100/...
  about: Monkey D. Luffy is the main protagonist...
  voiceActors:
    - name: Mayumi Tanaka
      image: ...
\`\`\`

## News
Get latest anime news and updates.

**Endpoint**

\`GET /news\`

**Example**

\`\`\`bash
curl http://localhost:3030/api/v1/news
\`\`\`

**Response Schema**

\`\`\`yaml
success: true
data:
  - id: news-123
    title: "New Season Announcement: Attack on Titan Final Season"
    image: https://cdn.noitatnemucod.net/thumbnail/300x400/100/...
    description: The final season of Attack on Titan...
    uploadedAt: 2 days ago
    category: Anime News
\`\`\`
