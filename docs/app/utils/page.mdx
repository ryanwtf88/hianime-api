# Utilities

## Random Anime

Get a random anime recommendation.

**Endpoint**

\`GET /random\`

**Example**

\`\`\`bash
curl http://localhost:3030/api/v1/random
\`\`\`

**Response Schema**

\`\`\`yaml
success: true
data:
title: Attack on Titan
alternativeTitle: Shingeki no Kyojin
id: attack-on-titan-112
poster: https://cdn.noitatnemucod.net/thumbnail/300x400/100/...
episodes:
sub: 25
dub: 25
eps: 25
type: TV
duration: 24m
\`\`\`

## Watch2Gether

Get available Watch2Gether rooms.

**Endpoint**

\`GET /watch2gether\`

**Example**

\`\`\`bash
curl http://localhost:3030/api/v1/watch2gether
\`\`\`

**Response Schema**

\`\`\`yaml
success: true
data:

- id: room-id-1
  title: One Piece Episode 1120
  episodeId: one-piece-100?ep=1120
  users: 5
  createdAt: 2024-01-01T10:00:00Z
  \`\`\`

---

## Watch2Gether Player

Get player page for a Watch2Gether room.

**Endpoint**

\`GET /watch2gether/player/:id\`

**Parameters**

| Parameter | Type   | Required | Description |
| --------- | ------ | -------- | ----------- |
| id        | string | Yes      | The room ID |

**Example**

\`\`\`bash
curl http://localhost:3030/api/v1/watch2gether/player/room-id-1
\`\`\`

**Response**

Returns an HTML page with the Watch2Gether player interface, allowing multiple users to watch anime together in sync.
