# Utilities

## Random Anime
Get a random anime recommendation.

**Endpoint**
\`GET /random\`

**Example**
\`\`\`bash
curl http://localhost:3030/api/v1/random
\`\`\`

---

## Watch2Gether
Get Watch2Gether rooms.

**Endpoint**
\`GET /watch2gether\`

**Example**
\`\`\`bash
curl http://localhost:3030/api/v1/watch2gether
\`\`\`

## Watch2Gether Player
Get player information for a Watch2Gether room.

**Endpoint**
\`GET /watch2gether/player/:id\`

**Example**
\`\`\`bash
curl http://localhost:3030/api/v1/watch2gether/player/12345
\`\`\`
